TorchScript Benchmarks

Reproducing

These results are created via this benchmark script and run nightly on the latest commit of PyTorch with this runner. The CSV results are pushed to this GitHub repo.

Bisecting a Performance Regression

Use the benchmark script.

Adding a New Benchmark

  1. Add your test to the benchmark script. See the README for details.
  2. Run the benchmark runner.
  3. Add the name of the test to to script.js.
